scrape-itebooks
==============================

prerequisite
------------

-	install python3
-	install pip

after get a copy
----------------

- `pip3 install pyyaml`
- `cd custom-utils`
- `python3 setup.py install`
- `pip3 install beautifulsoup4`
- `pip3 install --upgrade pip`
- `pip3 install SQLAlchemy`
- `pip3 install requests`
- `pip3 install html5lib`

get start
----------------
- `python3 main.py -d ~/Downloads/ebook`
